## 🎯 محتويات الملخص
يغطي هذا المرجع أهم المواضيع التقنية للمقرر بطريقة منظمة ومدعومة باختبارات تجريبية:
* **PTES Phases:** مراحل معيار تنفيذ اختبار الاختراق السبعة.
* **Test Strategies:** استراتيجيات الاختبار (الصندوق الأسود، الأبيض، الرمادي).
* **OSI Model:** طبقات نموذج الاتصال المفتوح السبعة.
* **TCP/IP & Protocols:** نموذج TCP/IP، ومقارنة شاملة بين بروتوكولات TCP و UDP و ICMP.
* **Ports:** أهم المنافذ واستخداماتها الأمنية.
* **Interactive Quizzes:** اختبارات قصيرة تفاعلية لكل قسم، واختبار شامل نهائي لتجربة مستواك.

## 🛠️ التقنيات المستخدمة
* **HTML5:** لبناء الهيكل وتنسيق النصوص ثنائية اللغة.
* **CSS3:** لتصميم واجهة حديثة، بطاقات تفاعلية، ووضع ليلي مريح للعين.
* **JavaScript:** لبرمجة التبويبات ونظام الاختبارات التفاعلية (حساب الدرجات، التحقق من الإجابات).
